家の査定依頼で注意することは？不動産会社はどこを見る？

2023.07.28

不動産売却の最初の一歩である物件（家・土地）の査定は、無料で依頼できます。

ですが、必要書類がそろっていないと評価が低くなってしまうことも……。準備不足で見積もりの精度が落ちてしまったら、査定にかけた時間がもったいないですよね。

今回の記事では、家の査定を依頼するときの注意点をご紹介します。あわせて、不動産会社がどこを見るのか、評価ポイントについても解説します。

家の査定前に準備すること

家の査定前に準備した方がいいことは、次の4つです。

  1. 住宅ローン残債の確認
  2. 必要書類をそろえる
  3. 家の状態を確認
  4. 家の相場を確認

ちなみに、家の査定前に片づけやハウスクリーニングをする必要はありません。査定額に影響はありませんが、購入希望者が内覧に来る前には部屋をきれいにしておくことをおすすめします。

1.住宅ローン残債の確認

住宅ローンの残債がいくらか調べておきましょう。なぜなら、家を売却する場合、基本的にはローンを返済し終えていることが条件になるからです。残りの住宅ローンが、家の査定額を上回っているケースは売却が難しくなります。

住宅ローンの残債を調べる方法で一番簡単なのは、金融機関のWebサイト上でチェックすることです。また、金融機関が郵送する返済予定表や残高証明書でも確認できます。

2.必要書類をそろえる

家の査定をスムーズに正確に行うために、必要書類を手元に準備しておきましょう。

  • 登記済権利証
  • 重要事項説明書
  • リフォーム関連書類（発注書など）
  • 建物の図面（平面図、パンフレットの間取り図）
  • 建築確認検査済証
  • 土地の実測図
  • 固定資産税納税通知書

すべての書類が必須ではないですが、家の状態を証明する資料が多いほど、査定の精度が上がります。リフォーム歴がある場合はプラスの査定につながる可能性もあるので、リフォーム内容がわかる書類を前もって準備しておくとよいでしょう。

3.家の状態を確認

家の状態をご自身で確認しておくことも重要です。

  • 築年数
  • リフォーム履歴
  • 土地の境界
  • 不具合（瑕疵）

「築年数」は古いほど評価が低くなります。

「家のリフォーム」は、外装のヒビ割れ舗装や屋根瓦の張り替え、基礎構造の塗装コーティングなど、定期的にメンテナンスを行ってきた物件は評価されます。「いつ」「どのような修繕をしたのか」を明確にしておきましょう。証明できる書類（発注書など）があるとベストです。

「土地の境界」は、一軒家（戸建て）で隣の土地との境界線があいまいな場合、不動産会社に伝える必要があります。場合によっては、公的な書類を請求したり、専門家に依頼して測定したりという工程が必要になることもあります。

「不具合（瑕疵）」は、物件の欠陥のことです。雨漏りする、シロアリ被害に遭ったことがある、ドアの建て付けが悪い…といった不具合は事前に知らせておかないと買主との裁判沙汰に発展するケースもあるので注意してください。

4.家の相場を確認

正確な査定は不動産のプロに依頼する必要がありますが、家がどのくらいの価格で売れるのか、相場を把握しておくことも肝心です。ある程度の目安を調査しておくことで、不動産会社から提示された査定額が相場と比較して妥当なのかを判断することができます。

インターネット上でできるAI査定では、簡単なアンケートに答えるだけで目安の金額がわかります。また、購入者向けの物件検索サイトで、自分の物件と近い条件の売り出し価格を調べるのもおすすめです。

いずれにせよ、あくまで目安の価格ですが、相場を知っておくことで査定額が高すぎる安すぎる場合に気づくことができます。査定価格の根拠についても質問しやすくなるので、事前に相場を知っておくことは重要です。

家を査定依頼するときの注意点

しっかり準備を終えたら、次はいよいよ査定依頼です。以下のことに注意してください。

  • 複数社に査定依頼する
  • 希望売却日と価格を伝える
  • アピールポイントを伝える
  • 不具合（瑕疵）を伝える

複数社に査定依頼する

査定依頼が1社だけなのはNGです。なぜなら、不動産会社によって査定基準は異なるため、査定額も異なります。複数社に依頼することで、物件の相場がわかります。

また、相場と比較して査定額が「高すぎる」もしくは「安すぎる」場合は注意してください。相場が高いことは一見いいように思えますが、あくまでも売却予想額です。実際は、高い価格では売れず、値下げで売り出すことになった上に、販売期間が長引いて損をすることもあります。

希望売却日と価格を伝える

査定依頼の際に、「いつまでに売りたいか」「いくら以上で売りたいか」を明確にすることで、希望に応じて対策をしてもらえます。

不具合（瑕疵）を伝える

物件の不具合は正直に伝えましょう。たとえば、次の内容に心当たりはないでしょうか？

  • シロアリ被害
  • 雨漏り
  • 壁のヒビ割れ
  • ガス管の漏れ
  • 物件での事件・事故
  • 土地の地盤沈下
  • 近隣の騒音

黙っていればわからないだろうと申告せずに契約した場合、「瑕疵担保責任」で裁判に発展するケースもあります。

つまり、売却した後で欠陥が見つかった場合でも、修理費用の請求や損害賠償を請求される可能性があるのです。最悪の事態を避けるために、マイナスポイントはしっかり伝えるようにしましょう。

家の査定はどこを見る？

不動産会社は家の査定でどこを見るのか気になりますよね。評価のポイントは次の4つです。

  • 物件の状態
  • 間取り
  • 周辺環境
  • 近隣トラブル

物件の状態

築年数は重要な要素で、築20年を超えた物件は価値が低いです。なぜなら、木造住宅の耐用年数は22年と定められているので、建物としての寿命が短いから。売却を検討しているなら、なるべく早めに動くことをおすすめします。

また、物件の劣化が激しい場合には評価が低くなります。雨漏りや家屋の傾きなどが見られる物件は、価値が下がってしまうでしょう。逆に、築年数が経っていても、家の傷みを早期発見し、メンテナンス・補修をしてきた家は査定アップにつながります。

間取り

生活導線を意識した、住みやすい間取りは高評価です。ファミリー向けであれば、リビングダイニングと複数の個室があるのが一般的な間取りで、需要が大きいです。
一方、広すぎるガレージやルーフバルコニー、ガラス張りのお風呂など、個性が際立つデザイナーズ住宅は売却しづらい傾向があります。

周辺環境

利便性の良い立地であれば高評価です。近くに、病院や学校、スーパーやコンビニなど暮らしに欠かせない施設があるエリアは、需要が高まります。

また、交通の便の良さも大きなポイントです。近くに駅やバス停がある、主要道路が近いエリアは人気があります。

また、土砂災害や洪水、津波、地震などの災害リスクが高いのはマイナス要因です。買い手が見つかりづらいため、査定価格は低めになります。

近隣トラブル

近隣でトラブルが発生していないかも評価に含まれます。子どもやペットの騒音、無断駐車、ゴミの不法投棄、悪臭・異臭などはマイナスポイントです。

特に、左右隣の家との関係は重要です。樹木が敷地内に侵入している、フェンスが境界線を越えているなど、トラブルがある場合は申告しておきましょう。

まとめ

今回は、家を査定依頼するときの注意点について解説しました。

査定前には、以下の準備をしておくと良いでしょう。

  • 住宅ローン残債の確認
  • 必要書類をそろえる
  • 家の状態を確認
  • 家の相場を確認

できる限りの必要書類をそろえておくことで、査定はスムーズに進み、精度も高くなります。物件に関するアピールポイントやマイナスポイントを正確に伝えることも大切です。しっかりと事前準備をして、査定にのぞみましょう。
